A comprehensive customer segmentation analysis using RFM (Recency, Frequency, Monetary) methodology to identify high-value customers, reduce churn, and drive revenue growth.

# 📊 Project Overview
This project analyzes 500,000+ transactions from 6,000+ customers using RFM analysis to segment customers into 11 actionable groups. The analysis identifies Champions worth $12M in revenue (69%), at-risk customers representing $2.1M in potential loss, and growth opportunities worth $3.5M in potential uplift.
# 🎯 Business Problem
E-commerce businesses struggle with:

Identifying their most valuable customers
Preventing customer churn
Optimizing marketing spend
Personalizing customer experiences

# 💡 Solution
Implemented RFM segmentation to:

✅ Segment 6,000+ customers into 11 actionable groups
✅ Identify top 20% customers generating 69% of revenue
✅ Flag 940+ at-risk customers worth $2.1M
✅ Create targeted strategies for each segment
✅ Build interactive Power BI dashboard for real-time insights

## 🧩 Customer Segmentation

| Segment              | RFM Criteria                 | Description                                                   |
|----------------------|------------------------------|---------------------------------------------------------------|
| Champions           | R ≥ 4, F ≥ 4, M ≥ 4           | Best customers: recent, frequent, and high spend             |
| Loyal Customers     | F ≥ 4, M ≥ 4                  | High-value repeat buyers, but less recent                    |
| Potential Loyalists | R ≥ 4, F ≥ 2, M ≥ 2           | Recently active with growing engagement                      |
| Recent Customers    | R ≥ 4, F ≤ 2                  | New customers with low frequency                             |
| Promising           | R ≥ 3, F ≥ 2, M ≥ 3           | Good potential for growth                                    |
| Need Attention      | R ≥ 3, F ≥ 2, M ≥ 2           | Showing signs of disengagement                               |
| About To Sleep      | R ≥ 2, F ≤ 3, M ≤ 3           | Risk of becoming inactive                                    |
| At Risk             | F ≥ 3, M ≥ 3, R < 3           | Previously high-value, now inactive                          |
| Cannot Lose Them    | F ≥ 4, M ≥ 4, R < 3           | Top customers becoming inactive — **urgent retention needed**|
| Hibernating         | R ≤ 2                         | Very inactive, low engagement                                |
| Lost                | R ≤ 2, F ≤ 2, M ≤ 2           | Lowest scores across all metrics                             |
